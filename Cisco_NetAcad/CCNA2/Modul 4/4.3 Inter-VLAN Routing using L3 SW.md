- Ve velkých sítích se nepoužívá router-on-a-stick -> Špatně se škáluje
- L3 SW:
	- Routuje hardwarově - je rychlejší než router
	- Je v distribuční vrstvě campus LAN
### Schopnosti L3 SW
- Routování mezi VLANy pomocí SVI
- Převod portu z L2 na L3 routed port (no switchport)
- Každá routovatelná VLAN musí mít vlastní SVI

### Vytvoření SVI
`interface vlan VLAN_ID`
- Funguje stejně jako router interface

### Konfigurace L3 SW
#### Postup
1. Vytvoření VLAN
2. Vytvoření SVI rozhraní
3. Nastavení access portů
4. Zapnutí IP routingu

- VLAN 10 -> LAN10
- VLAN 20 -> LAN20

### Kontrola spojení při inter-VLAN routingu
- Kontrola konfigurace na PC přes ipconfig
- Test komunikace pomocí pingu

### Routování na L3 SW
- Pokud mají být VLANy dostupné i mimo SW musí te použít statické nebo dynamické routování
- Používá se routed port
	- Vznikne příkazem no switchport
	- Chová se jako routed port
		- L3 rozhraní, nepatří do VLAN

### Konfigurace routování na L3 SW
#### Postup
1. Nastavit routed port
2. Povolit IP routing
3. Nastavit routing (OSPF)
4. Ověřit routování
5. Ověřit konektivitu
