### 5 fází zapínání switche:

1. **POST (Power-On-Self-Test)** - Test CPU, DRAM a části flash paměti
2. **Načtení boot loaderu** - Malý program v ROM
3. **Inicializace CPU** - Boot loader nastaví registry CPU 
4. **Inicializace souborového systému flash** - Připraví flash paměť pro práci
5. **Načtení IOS** - Boot loader najde a načte IOS image

---

### Příkaz ```boot system```

-  Switch se snaží automaticky spustit podle proměnné BOOT
-  Pokud není nastavena, hledá první spustitelný soubor v paměti
-  Na Catalyst 2960 je IOS obvykle ve složce pojmenované podle souboru

|                            Command |                Definice |
| ---------------------------------: | ----------------------: |
|                  ```boot system``` |            Main command |
|                       ```flash:``` |          Storage device |
| ```/c2960-lanbasek9-mz.150-2.SE``` | Path to the file system |
| ```/c2960-lanbasek9-mz.150-2.SE``` |           IOS file name |

---
### Indikátory LED na switchi

- Umožňují sledovat stav a výkon switche
- Zelená = OK, Oranžová = chyba, Vypnutá = bez napájení

1. **SYST** - Systém 
2. **RPS** - Redunantní napájecí zdroj
3. **STAT** - Stav switche
4. **DUPLX**- Duplexní režim portu
5. **SPEED** - rychlost portu
6. **PoE** - Napájení přes etherent

---
### Obnovení po selhání systému

- Boot loader umožňuje přístup k souborům v flash, pokud IOS nefunguje
##### Postup přístupu do boot loaderu:
1. Připojit PC na konzolový kabel
2. Odpojit napájení switche
3. Připojit napájení a držet 15 vteřin MODE button dokud SYST LED nezmodrá a pak nezzelená
4. Boot loader prompt: ```switch:```
5. Pomocné příkazy jsou ```Help``` a ```?```
6. Inicializace flash: ```flash_init```
7. Změna BOOT proměnné a spuštění nového IOS:
		```BOOT=flash:c2960-lanbasek9-mz.150-2.SE8.bin 
		set
		boot```

### Správa SW
- Pro vzdálenou správu musí mít SW SVI s IPv4 nebo IPv6 adresou
- Pro správu mimo lokální síť je potřeba nastavit **default gateway**

---
### Příklad konfigurace SVI

- Výchozí správa: VLAN 1 (bezpečnější použít VLAN jinou než 1, např. VLAN 99).
- Nastavení IPv4 a IPv6 adresy:
	`VLAN 99 IPv4: 172.17.99.11/24 IPv6: 2001:db8:acad:99::1/64`
- Poznámka: SVI bude “up/up” až po vytvoření VLAN a připojení zařízení k portu.
