### Telnet
- Používá TCP port 23
- Přenáší uživatelská jména, hesla, data v plaintextu
- Lze snadno odposlechnout
---
### SSH
- TCP port 22
- Přenos je šifrovaný, data jsou bezpečná
- SSH nahrazuje Telnet pro vzdálenou správu
---
### Ověření podpory SSH switchem
- Na Catalyst 2960 je potřeba IOS s kryptografickými funkcemi
- `show version` - Pro kontrolu IOS
- `show ip ssh` - Pro ověření SSH
---
### Konfigurace SSH switchem
##### 1. krok
`show ip ssh` - Ověří jestli SSH SW vůbec podporuje
##### 2. krok
`ip domain-name cisco.com` - Konfiguruje IP doménu sítě v globálním režimu
##### 3. krok
`crypto key generate rsa` - Generování RSA klíčů
##### 4. krok
`username admin cesret ccna` - Konfigurace Admin autentifikace
##### 5. krok
`line vty 0 15` - Konfigurace VTY linek
`transport input ssh`
`login local`
`exit`
##### 6. krok
`ip ssh version 2` - Zapnutí SSH verze 2
