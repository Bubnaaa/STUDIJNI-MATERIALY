### Základní nastavení routeru
- Jsou podobné switchům (IOS, Módy konfigurace,...) 

```
hostname R1
enable secret class
```

```
line console 0
password cisco
login
line vty 0 4
password cisco
login
```

```
service password-encryption
```

```
copy running-config startup-config
```

---
### Dual stack topologie
- Routery podporují IPv4 i IPv6
- Oproti switchům podporují LAN i WAN

---
### Konfigurace router interface
- Každý interface musí být:
	- 1. Konfigurované s IP adresou 
		```
		ip address 192.168.10.1 255.255.255.é
		ipv6 address 2001:db8:acad:1::1/64
		```
	- 2. Aktivní
	 ```
		 no shutdown
	```
---
### IPv4 Loopback rozhraní
- Loopback je logické, softwarové rozhraní, nikdy nepřipojené k fyzické síti
- Výhody:
	- Testovaní routeru a routing procesů
	- Simulace dalších sítí v laboratoři
	- Vždy UP pokud router funguje
- Konfigurace:
```
interface loopback 0
	ip adress 10.0.0.1 255.255.255.0
```
- Loopbacků může být více, každé s unikátní IPv4 adresou
